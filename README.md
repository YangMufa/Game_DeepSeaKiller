### 基本要求
* Java: 1.8
* IDEA: 2021.3
* Mysql: 无

### 示例博客：http://www.muyiafa.com（开发中）

### 功能点
* 战舰炮弹Y向下移动
* 鱼雷Y向上移动
* 深水炸弹Y向上移动
* 鱼雷潜艇X向右移动
* 深水炸弹潜艇X向右移动
* 侦查潜艇X向右移动
