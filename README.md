优点：减少逻辑性，便于入门学习...
缺点：玩法单一...
体量：核心代码约 650 行

### 开发环境
* Java: 1.8
* IDEA: 2021.3

### 示例博客：http://www.muyiafa.com

### 功能点
* 战舰炮弹Y向下移动
* 鱼雷Y向上移动
* 深水炸弹Y向上移动
* 鱼雷潜艇X向右移动
* 深水炸弹潜艇X向右移动
* 侦查潜艇X向右移动
* 战舰击中潜艇加分
* 潜艇击中战舰减命
